# MealsCards — Gestionnaire de repas

Organise tes repas selon ce que tu as dans le frigo. Gère ton stock, planifie ta semaine, filtre tes repas intelligemment selon tes calories restantes, et génère ta liste de courses.

## Technologies

- [Vite](https://vitejs.dev/)
- [React](https://react.dev/)
- [TypeScript](https://www.typescriptlang.org/)
- [shadcn/ui](https://ui.shadcn.com/)
- [Tailwind CSS](https://tailwindcss.com/)
- [Supabase](https://supabase.com/)

## Démarrage

```sh
# Installer les dépendances
npm install

# Lancer le serveur de développement
npm run dev
```

L'application sera disponible sur `http://localhost:8080`.

## Build

```sh
npm run build
```
